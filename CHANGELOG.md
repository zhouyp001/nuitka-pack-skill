# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-07-12

### Added

- `SKILL.md` — complete Nuitka packaging skill covering pre-compilation checks,
  compilation command generation, progress monitoring, post-build verification,
  and a common-issues lookup table.
- `scripts/monitor_cpu.py` — CPU and memory monitor that beeps when CPU usage
  drops below a threshold, useful for detecting long-running builds' completion.
- Bilingual documentation (`README.md` in English, `README_CN.md` in Chinese).
- MIT license.
